# Blocked Tasks Analysis

This document tracks tasks that are currently blocked and cannot proceed due to dependencies, issues, or missing information.

## Blocked Task Format

For each blocked task, document:

- **Task ID**: Unique identifier
- **Epic/Story**: Parent epic and story
- **Description**: What the task is trying to accomplish
- **Block Reason**: Why it's blocked
- **Root Cause**: Underlying issue
- **Dependencies**: What needs to happen first
- **Potential Solutions**: Possible ways to unblock
- **Blocked Since**: Date when task became blocked
- **Last Reviewed**: Date of last review
- **Status**: Active, Resolved, Escalated

## Resolution Tracking

When a blocked task is resolved:

1. Move task back to `todo_progress.json` with updated status
2. Document resolution in this file
3. Add lessons learned to `completed_tasks.json`
4. Update any related documentation
