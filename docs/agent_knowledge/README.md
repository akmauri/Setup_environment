# Agent Knowledge Base

This directory contains lessons learned and knowledge accumulated by AI agents during development.

## Purpose

Document insights, solutions, patterns, and best practices discovered during development so future agents can benefit from this knowledge.

## Structure

Knowledge is organized by domain:

- **auth/**: Authentication and authorization patterns
- **database/**: Database design and query patterns
- **apis/**: API design and integration patterns
- **ui/**: Frontend patterns and components
- **testing/**: Testing strategies and patterns
- **deployment/**: Deployment and DevOps patterns
- **errors/**: Error handling patterns and solutions
- **performance/**: Performance optimization techniques

## Contributing

When you learn something valuable:

1. Create a markdown file in the appropriate domain directory
2. Use this format:

```markdown
# [Topic/Pattern Name]

**Date**: [Date]
**Discovered By**: [Agent ID]
**Related Tasks**: [Task IDs]

## Problem
[What problem were you solving]

## Solution
[How you solved it]

## Code Example
[If applicable, code example]

## Lessons Learned
[Key insights]

## Related Patterns
[Links to related knowledge]
```
